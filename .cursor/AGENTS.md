# Cursor Tooling Rules

## Module Context

- 이 폴더는 Cursor 전용 설정과 보조 도구를 관리한다.
- `mcp.json`은 MCP 서버 연결 설정을 담당한다.
- `skills/`는 반복 작업용 Cursor 스킬을 보관한다.

## Tech Stack & Constraints

- 설정 형식: JSON, Markdown
- 현재 MCP 설정에는 `context7` 서버가 등록되어 있다.
- 이 영역은 앱 런타임 코드가 아니라 개발 보조 설정을 위한 공간이다.

## Implementation Patterns

- MCP 설정은 실제 사용 중인 서버만 남긴다.
- 스킬은 하나의 목적에 집중시키고, 과도하게 범용적인 이름을 피한다.
- 스킬 설명에는 무엇을 하는지와 언제 써야 하는지를 함께 적는다.
- 프로젝트 규칙을 생성하거나 갱신하는 스킬은 실제 저장소 구조를 기준으로 동작해야 한다.

## Testing Strategy

- JSON 수정 후에는 문법이 유효한지 먼저 확인한다.
- 스킬 수정 후에는 설명과 본문이 현재 저장소 구조와 충돌하지 않는지 확인한다.
- 존재하지 않는 파일 경로나 도구를 전제로 하는 설정은 남기지 않는다.

## Local Golden Rules

- `.cursor` 설정에 비밀값을 직접 넣지 않는다.
- 앱 런타임 규칙과 Cursor 도구 규칙을 같은 파일에 섞지 않는다.
- 내장 스킬과 프로젝트 스킬의 역할을 구분한다.
- 저장소 구조가 바뀌면 관련 스킬과 MCP 설정도 함께 점검한다.
